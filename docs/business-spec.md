## 1. Executive Summary

**Nexy** (nexy.life) — web-приложение для пар, которое помогает партнёрам обнаружить общие интимные предпочтения через анонимный matching. Несовпавшие желания **никогда** не раскрываются партнёру, что снимает барьер стыда и страха отказа.

Бизнес-модель: freemium с подпиской $6.99/мес или $49.99/год. Web-first подход обходит ограничения App Store/Google Play на sexual content.

---

## 2. Проблема

### Барьеры коммуникации в парах

- **67% пар** испытывают трудности с обсуждением сексуальных предпочтений (исследования Kinsey Institute)
- **Страх отказа** — главная причина: «А вдруг партнёр подумает, что я странный/странная?»
- **Информационная асимметрия** — партнёры не знают, что другой может быть открыт к тому же
- **Нехватка языка** — люди буквально не знают, как назвать и сформулировать желания
- **Табуированность** — тема остаётся стигматизированной, особенно в русскоговорящем сегменте

### Почему существующие решения не работают

| Подход | Проблема |
|--------|----------|
| Разговор | Стыд, страх, неумение формулировать |
| Текстовые анкеты (Spicer, Kindu) | Скучно, нет визуала, нет адаптации |
| Секс-терапевт | Дорого ($100-300/сессия), стигма |
| Порно | Не про коммуникацию, искажает ожидания |

---

## 3. Решение

Nexy решает проблему через три механизма:

### 3.1 Анонимный matching

Каждый партнёр отвечает на вопросы **независимо**. Система показывает совпадения, но **никогда не раскрывает** несовпавшие желания. Это полностью устраняет страх отказа.

### 3.2 Визуальный подход

Вместо текстовых вопросов — **карточки с иллюстрациями** (comic book sketch style). Свайп-механика привычна из dating apps, снижает когнитивную нагрузку.

### 3.3 Адаптивный AI

- **357 сцен** в 33 категориях, подбираемые алгоритмом (70% exploitation / 30% exploration)
- **Body Map** — интерактивная карта тела для тактильных предпочтений
- **AI-аватар партнёра** — чат-бот, говорящий от лица партнёра на основе его реальных предпочтений
- **16 сексуальных архетипов** — для самопознания и взаимопонимания

## 4. Целевая аудитория

### 4.1 ICP (Ideal Customer Profile)

**Пары в отношениях от 1 года**, возраст 25-40, городские, digital-native, интересуются саморазвитием и качеством отношений.

### 4.2 Сегменты

| Сегмент | Описание | Доля | Conversion rate |
|---------|----------|------|----------------|
| **Curious Couples** | Пары, желающие «добавить остроты» | 50% | Средний |
| **Communicators** | Открытые пары, ищущие инструмент для разговора | 25% | Высокий |
| **Stuck Couples** | Пары в рутине, где секс стал проблемой | 15% | Высокий (pain-driven) |
| **Explorers** | Опытные пары, ищущие новое (kink/BDSM curious) | 10% | Самый высокий |

### 4.3 Персоны

**Аня, 29** — в отношениях 3 года. Секс стал рутинным. Хочет попробовать новое, но боится, что парень «не так поймёт». Читает Cosmopolitan, слушает подкасты про отношения.

**Дима, 32** — женат 5 лет. Есть фантазии, о которых стесняется сказать жене. Считает, что «у нас всё нормально», но хотел бы больше. Технически грамотный, привык к приложениям.

**Марина и Олег, 35** — пара со стажем, прошли курс терапии. Ищут конкретный инструмент для продолжения диалога. Готовы платить за качество.

### 4.4 География (приоритет запуска)

1. **RU** — русскоговорящие пары (Россия, СНГ, эмигранты). Продукт билингвален, но primary market — RU
2. **EN** — англоговорящий рынок для масштабирования (US, UK, CA, AU)

## 8. Unit Economics (оценки)

### 8.1 Ключевые метрики

| Метрика | Оценка | Обоснование |
|---------|--------|-------------|
| **ARPU** (Free) | $0 | Нет рекламы |
| **ARPU** (Premium monthly) | $6.99 | Месячная подписка |
| **ARPU** (Premium yearly) | $4.17/мес | Годовая подписка |
| **Blended ARPU** (paid users) | ~$5.50/мес | Предполагаемый микс 40% monthly / 60% yearly |
| **Free-to-Paid conversion** | 5-8% | Benchmark для freemium apps (dating/wellness) |
| **Monthly churn** (monthly plan) | 8-12% | Типичный для subscription apps |
| **Monthly churn** (yearly plan) | 2-4% | Annual lock-in |
| **Average subscription life** | 6-10 мес | Зависит от плана |
| **LTV** (paid user) | $35-55 | ARPU × lifetime |

### 8.2 CAC цели

| Канал | Целевой CAC | Примечание |
|-------|-------------|------------|
| Organic/SEO | $0-2 | Контент-маркетинг, блог |
| Viral (invite partner) | $0 | Встроенный growth loop |
| Social media | $3-8 | TikTok, Instagram (ниша — отношения) |
| Paid (Meta/Google) | $10-25 | Ограничен для sexual content |
| Influencer | $5-15 | Секс-блогеры, психологи |

### 8.3 LTV:CAC

**Целевое соотношение:** ≥ 3:1

При LTV $45 и CAC $10-15 → LTV:CAC = 3:1 — 4.5:1 ✓

### 8.4 Payback Period

При blended ARPU $5.50/мес и CAC $12 → **payback ~2.2 месяца**

### 10.4 Встроенные Growth Loops

```
User A регистрируется
    → Проходит онбординг (получает ценность solo)
        → Приглашает партнёра (User B) для matching
            → User B регистрируется (бесплатно)
                → Оба получают совпадения (aha moment)
                    → User A/B рассказывают друзьям
                        → Новая пара регистрируется
```

**Ключевая особенность:** приглашение партнёра — **не опциональная фича**, а core flow продукта. Каждый пользователь = потенциальный реферал.

## 15. Ключевые решения и обоснования

| Решение | Обоснование |
|---------|-------------|
| **Web, не mobile** | Обход App Store policy на sexual content; быстрая итерация; нет 30% комиссии Apple |
| **Freemium, не paywall** | Solo-value бесплатен → пользователь чувствует пользу → приглашает партнёра → конвертируется |
| **$6.99/мес** | Ниже конкурентов, на уровне развлекательных подписок, низкий порог входа |
| **Comic book illustrations** | Не фото (legal risk), не текст (скучно). AI-generated = масштабируемо |
| **RU-first** | Менее конкурентный рынок, чем EN. Отсутствие качественных RU-аналогов |
| **Privacy-first architecture** | Не маркетинг, а техническое решение (RLS, раздельные запросы). Trust = retention |
| **Adaptive progression** | Против «одноразовости» — контент усложняется, новые темы открываются |
